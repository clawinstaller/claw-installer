## Posting Order (Recommended)

1. **GitHub Discussion first** — Get signal from actual OpenClaw users
2. **Wait 2-3 days** — Gather feedback, iterate
3. **Reddit post** — Broader selfhosted audience
4. **Twitter thread** — After you have screenshots/demo GIF

## Before Posting

- [ ] Generate demo screenshots or GIF
- [ ] Verify all technical claims (Node version, issue percentages)
- [ ] Have working "Install" button (awkward to announce an installer that can't install)
- [ ] Prepare to respond to comments within 2 hours of posting

## Metrics to Track

- GitHub Discussion: 👍 reactions, comment count, sentiment
- Reddit: upvote ratio, comment engagement, cross-posts
- Twitter: impressions, replies, quote tweets, follows
